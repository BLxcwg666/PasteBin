# PasteBin
一个剪贴板  
写着玩的 XD

## 环境
- Node.JS ≥ 18
- Next.JS 15
- Redis ≥ 7.0
- pnpm

## 配置
将 `.env.example` 重命名为 `.env`，修改之后保存  
后端请见 https://github.com/BLxcwg666/PasteBin-API

## 运行
克隆本仓库  
运行`pnpm i`和`pnpm build`和`pnpm start`  
后续会做 docker 镜像，尽请期待
